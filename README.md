# Malware File Scanner Pro

A modern full-stack cybersecurity web application that analyzes uploaded files for suspicious behavior and generates detailed security reports.

## Features

* Drag & Drop File Upload
* File Metadata Analysis
* MD5, SHA1, and SHA256 Hash Generation
* Dangerous Extension Detection
* Double Extension Detection
* Suspicious Filename Detection
* JavaScript Threat Analysis
* Macro Indicator Detection
* Risk Scoring System
* Security Report Generation
* PDF Report Export
* Dashboard Analytics
* Scan History Storage
* Threat Intelligence Feed
* Responsive Cybersecurity-Themed UI

## Tech Stack

### Frontend

* React
* Tailwind CSS
* Framer Motion
* Chart.js

### Backend

* Node.js
* Express.js

### Storage

* Local Storage
